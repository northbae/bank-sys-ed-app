# Bank System

Проект представляет собой систему банковских операций с использованием **Java** и **Spring Boot**.

Система разделена на три независимых модуля, взаимодействие с которыми происходит через REST API. Документация и тестирование эндпоинтов реализованы через **Swagger (OpenAPI)**.

## Структура проекта

Проект состоит из трех основных модулей:

1.  **user-service** (модуль `user`)
    *   Регистрация и аутентификация клиентов.
    *   Управление профилем пользователя (KYC, личные данные).
2.  **account-service** (модуль `account`)
    *   Открытие и закрытие банковских счетов.
    *   Проверка баланса.
    *   История операций по счету.
3.  **card-service** (модуль `card`)
    *   Выпуск пластиковых/виртуальных карт.
    *   Привязка карт к счетам.
    *   Управление статусом карты (блокировка/активация).

## Технологический стек

*   **Java** 17+
*   **Spring Boot 3.x** (Web, Data JPA)
*   **PostgreSQL**
*   **Maven** 
*   **SpringDoc OpenAPI**

## Запуск и Установка

Так как проект является монорепозиторием, каждый модуль запускается как отдельное Spring Boot приложение.

### Предварительные требования
*   Установленный JDK.
*   Maven.
*   Настроенная БД (если не используется H2 in-memory).

### Запуск модулей

Вы можете запустить каждый сервис в отдельном терминале:

**1. Запуск User Service**
```bash
cd user
mvn spring-boot:run
```
**2. Запуск Account Service**

```Bash

cd account
mvn spring-boot:run
```
**3. Запуск Card Service**

```Bash

cd card
mvn spring-boot:run
```
## API Documentation (Swagger)
После запуска сервисов, Swagger UI доступен по следующим адресам:

Модуль	Порт (по умолчанию)	Swagger URL

User	8082	http://localhost:8082/swagger-ui.html

Account	8081	http://localhost:8081/swagger-ui.html

Card	8083	http://localhost:8083/swagger-ui.html

## Взаимодействие
Сервисы общаются друг с другом по REST API

Основной поток:

Создание пользователя -> (user-service)

Создание счета для пользователя -> (account-service)

Выпуск карты к счету -> (card-service)
